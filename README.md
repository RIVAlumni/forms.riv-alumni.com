# RIVA Forms

## Local Development

### Prerequisites

Install [yarn](https://yarnpkg.com/getting-started/install).

### Initial Setup

Run the following shell command to install packages.

```bash
yarn install
```

### Running Locally

Run the following shell command to start the development server.

```bash
yarn dev
```

### Adding Dependencies

Run `yarn add` as per usual.
